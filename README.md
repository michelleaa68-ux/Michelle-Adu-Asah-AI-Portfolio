# AI-Portfolio
Over the semester I learned how to work with text data using NLP, including cleaning, processing, and analyzing language in a structured way. I also gained experience with machine learning and some deep learning concepts, and became more comfortable coding in Python using both Google Colab and VS Code. I also worked with data science workflows and basic database design, which helped me understand how data is stored and used in real systems.

My biggest project is NewsBot 2.0, an NLP system that can classify news articles, analyze sentiment, detect topics, generate summaries, and identify the language of the text.

### NewsBot 2.0 — NLP News Analysis System  
This project is an NLP-based system that analyzes news articles by classifying them, detecting sentiment, identifying topics, generating summaries, and detecting language.
